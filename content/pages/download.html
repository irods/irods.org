Installation Examples

sudo apt-get install irods-server irods-database-plugin-postgres
or 
sudo yum install epel-release

Upgrade Examples

sudo apt-get install irods-server irods-database-plugin-postgres
or 
sudo yum update irods-server irods-database-plugin-postgres
